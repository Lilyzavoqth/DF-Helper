# DF Helper
An Addon to help with DFing enemies that are in the sky or that have spawning in timers.

Morfos timer only works if there is a player in the room when it spawns in. Swing at 20 or below.

### Requirements
v0.3.7.0+ of the [psobbaddonplugin](https://github.com/Solybum/psobbaddonplugin)

### Installation
1. [Download](https://github.com/Lilyzavoqth/DF-Helper/archive/refs/heads/main.zip)
2. Place the "DF Helper" folder inside the "addons" folder inside your Ephinea installation.
